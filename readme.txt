== Description ==

SGS Social Sharing Buttons is a lightweight (6kb) Social Sharing plugin that is used to add a fixed social sharing buttons to your WordPress site. After installation, no configuration is required. It seamlessly works with AMP and non AMP pages and appears on all  site's pages.


== Installation ==

1. log in to your WordPress Admin area
2. Go to "Plugins > Add New" from the left-hand menu
3. In the search box type "SGS Social Sharing Buttons"
4. From the search result you will see "SGS Social Sharing Buttons" click on "Install Now" to install the plugin
5. A popup window will ask you to confirm your wish to install the Plugin.
6. Click "Proceed" to continue the installation. The resulting installation screen will list the installation as successful or note any problems during the install.
7. Click "Activate Plugin" to activate it.

== Frequently Asked Questions ==
**Does it supports Accelerated Mobile Pages, aka, AMP?**
Yes, it perfectly works on AMP and non-AMP WordPress site pages.

**Is the plugin mobile friendly?**
Yes, it is responsive and it fits well on mobile devices to facilitate sharing.

**Does it support non-English Langauage**
Yes, it works for all langages and script types.
